Welcome to RFEM/RSTAB Webservices documentation!Â¶

Web service is a programmable interaction system for RFEM/RSTAB. With the help of this technology, RFEM/RSTAB provides a server service that can be reached locally or via a network. It is possible to send requests and get responses from RFEM/RSTAB with this client-server communication.

Note

Detailed information can be found in our Online Manual.

You are very welcome to contribute to RFEM/RSTAB Webservices with new documents and examples by sending pull requests through GitHub Pulls.

For any errors or issues please contact us via support@dlubal.com or create an issue via GitHub Issues.
