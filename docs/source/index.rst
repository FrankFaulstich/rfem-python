Welcome to RFEM/RSTAB Webservices documentation!
==================================================
Web service is a programmable interaction system for RFEM/RSTAB. 
With the help of this technology, RFEM/RSTAB provides a server service that can be reached locally or via a network. 
It is possible to send requests and get responses from RFEM/RSTAB with this client-server communication.

.. note:: 

   Detailed information can be found in our `Online Manual <https://www.dlubal.com/en/downloads-and-information/documents/online-manuals/rfem-6-rstab-9-webservice-api>`_.
   
   You are very welcome to contribute to RFEM/RSTAB Webservices with new documents and examples by sending pull requests through `GitHub Pulls <https://github.com/Dlubal-Software/RFEM_Python_Client/pulls>`_.
   
   For any errors or issues please contact us via support@dlubal.com or create an issue via `GitHub Issues <https://github.com/Dlubal-Software/RFEM_Python_Client/issues>`_.
