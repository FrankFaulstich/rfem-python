Overview
========================

Web service is a programmable interaction system for RFEM/RSTAB. With the help of this technology, RFEM/RSTAB provides a server service that can be reached locally or via a network. 
It is possible to send requests and get responses from RFEM/RSTAB with this client-server communication.

.. note:: 

   More information can be found in our `Website <https://www.dlubal.com/en/solutions/online-services/webservice-and-api>`_.
   
   For any errors or issues please contact us via support@dlubal.com or create an issue via `GitHub Issues <https://github.com/Dlubal-Software/RFEM_Python_Client/issues>`_.
